# Sistema de Gerenciamento de Pedidos em C++

Projeto desenvolvido para a disciplina de Programação Orientada a Objetos da Universidade Federal de São Carlos (UFSCar).

## Objetivo

Implementar um sistema de gerenciamento de pedidos utilizando os principais conceitos de Programação Orientada a Objetos (POO).

## Funcionalidades

- Cadastro de produtos
- Cadastro de clientes físicos
- Cadastro de clientes jurídicos
- Gerenciamento de pedidos
- Associação de pedidos a clientes
- Controle de itens dos pedidos

## Conceitos Aplicados

- Encapsulamento
- Herança
- Polimorfismo
- Classes abstratas
- Sobrescrita de métodos (`override`)
- Organização modular em múltiplos arquivos
- Tratamento de exceções

## Tecnologias Utilizadas
- C++
- CMake
- Programação Orientada a Objetos
  
## Como Executar
Windows (CLion)
- Abrir o projeto no CLion e executar normalmente.
